# Day 40 – AI Assistant Builder

## Objective

Build a complete AI-powered Revision Planner using Claude AI Assistant Builder.

---

## What I Built

Today's project is a Revision Planner that helps students prepare for exams by generating personalized revision schedules.

The assistant collects:

- Exam information
- Subjects
- Study hours
- Weak topics
- Additional notes

It then creates a structured revision timeline using Claude AI.

---

## Features

- Interactive multi-step wizard
- Dynamic revision planning
- Timeline visualization
- Strategy recommendations
- Documentation section
- Responsive interface

---

## Skills Learned

- AI Assistant Builder workflow
- Prompt engineering
- System prompt design
- JSON-based AI outputs
- User experience design
- Error handling

---

## Challenges

During testing, the schedule generation occasionally produced a response parsing error due to inconsistent AI output formatting. This highlighted the importance of validating structured AI responses and implementing robust JSON parsing.

## Outcome

Successfully built and tested an AI-powered Revision Planner while learning how Claude Assistant Builder creates interactive AI applications.
