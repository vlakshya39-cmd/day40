# 📚 Revision Planner AI Assistant

An intelligent revision planning assistant built using **Claude AI Assistant Builder**.

The application helps students create personalized study schedules by collecting exam information, subjects, available study hours, and learning preferences through a guided multi-step workflow.

---

## 🚀 Features

- Multi-step revision planning wizard
- Personalized study schedule generation
- Subject prioritization with weak-topic highlighting
- Interactive revision timeline
- Study strategy recommendations
- Responsive modern UI
- Documentation panel explaining system design
- Error handling and schedule validation
- Claude API integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Claude AI Assistant Builder
- Anthropic Claude API

---

## 📌 How It Works

1. Enter exam details.
2. Add subjects to study.
3. Mark difficult subjects.
4. Provide additional notes or syllabus.
5. Generate a personalized revision schedule.
